Here is the complete, production-ready code for your Futuristic Developer Chrome New Tab Page.

This is implemented as a single, self-contained HTML file (index.html) using modern HTML5, Tailwind CSS (for structure), dynamic CSS variables for custom glow/glassmorphism effects, and highly optimized vanilla ES6+ JavaScript. It features a interactive background canvas animation, fully-functional local storage, live weather geolocator, a daily checklist with automatic midnight IST reset, and a searchable/customizable bookmark manager.

Features & Architecture Implemented:
Cyberpunk Minimal Aesthetic: Deep #050507 and #0A0A0F dark environment featuring a premium frosted glassmorphism UI card structure. Accentized with neon glows utilizing hexes representing your chosen Cyan (#00E5FF), Purple (#7C4DFF), and Success (#00FF95) theme guidelines.
Interactive Background Network: Built on a highly-optimized dynamic HTML5 Canvas backend tracking grid coordinates and rendering interactive particle configurations that establish fading network nodes on trajectory overlap.
Advanced Weather Processing: Integrates latitude and longitude coordinates fetched directly via navigator.geolocation linked with open-source reverse geocoding via OpenStreetMap API. Weather metrics are populated dynamically on load using the Open-Meteo API (zero API key required).
Time & System Simulator: The date and time metrics are simulated dynamically in modern digital monospace fonts (via JetBrains Mono) matching the requested styling structure.
Stateful Bookmarks Organizer: Comes out of the box pre-loaded with developer default resources split into collapsible categories ("Development", "Learning", "AI Resources"). Users can dynamically add, delete, collapse, and create categorization directories stored reliably inside localStorage.
Robust Task Queue Manager with Automatic Reset (00:00 IST):
Dynamic, interactive checklist tracking execution targets.
Built-in timezone compliance architecture mapping exact dates against Asia/Kolkata (Indian Standard Time).
If a new day changes, current uncompleted or completed tasks are archived on initialization into a backup registry system accessible through the History modal, refreshing the active directive checklist layout for the current target cycle.
